# Today I Learned (TIL)

## 📝 작성 방법

```
# YY.MM.DD TIL

## 주제: [오늘 배운 주요 내용]

### 학습 내용

#### 핵심 개념 
   - 상세 설명
   - 코드 예시

### 느낀 점

- 오늘 배운 내용에 대한 개인적인 생각이나 소감

### 질문 목록

1. 질문1
2. 질문2

### 참고 자료:

- [관련 문서, 책, 웹사이트 링크]
```

## 🎯 TIL 목적

> 학습 우선순위를 정하고 그 날의 학습한 내용을 기록으로 남기기 위해

"설명할 수 없다면 제대로 아는 것이 아니다" 라고 생각합니다. 학습한 내용을 나만의 방식으로 정리해 확실히 이해하고 정보를 공유하고자 합니다.
